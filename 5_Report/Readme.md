  # SWOT Analysis
  
  ## Strengths
  
  * Keeps your thoughts organized and make them apprehensive.
  * Stress Relieving .
  * Allows yourself to self-reflect .
  * Easy to design.
  
  ## Weakness
  * Writing diarires can be time consuming.
  * No password protection.
